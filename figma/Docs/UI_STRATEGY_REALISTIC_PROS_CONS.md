# UI 전략별 장단점 (현실적 분석)

**중요**: Custom CSS 90% 전략은 SLDS와 독립적이므로 **Salesforce 업데이트 영향을 거의 받지 않습니다**.

## 수정된 픽셀 퍼펙트 전략 단점

### ❌ 잘못된 설명:
> "Salesforce 업데이트 시 호환성 문제"

### ✅ 실제 문제:
1. **CSS 코드량 증가** (1000+ 줄)
2. **디자인 변경 시 수동 작업** (20개 파일 수정)
3. **접근성 직접 구현** (4시간 추가)
4. **신규 개발자 학습 경사** (2주 온보딩)
5. **Salesforce 생태계 이질감**

## 실제 사례 비교

### 시나리오: "모든 버튼 색상 변경"

**픽셀 퍼펙트**: 20개 파일 수동 수정 (30분)
**하이브리드**: 1개 파일만 수정 (5분)
**SLDS 표준**: 테마 설정 클릭 (1분)

## 비교표 (수정)

| 지표 | 픽셀 퍼펙트 | 하이브리드 | SLDS 표준 |
|------|-----------|------------|----------|
| Salesforce 업데이트 영향 | ✅ 없음 | ⚠️ 낮음 | ⚠️ 중간 |
| CSS 코드량 | 1000+ 줄 | 400줄 | 100줄 |
| 디자인 변경 | 30분 | 5분 | 1분 |
| 접근성 구현 | 4시간 | 부분 자동 | 자동 |
